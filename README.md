# FDA Food Recall Extractor

Utilizing the openFDA Food Enforcement API
- Endpoint: `https://api.fda.gov/food/enforcement.json`
- Query Logic: Targeting a single result (`limit=1`) to deep-dive into the structure of an enforcement report.
- Key Data Point: This project focuses on identifying the root cause of recalls (e.g., *Listeria monocytogenes* contamination).
